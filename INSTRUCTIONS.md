# 使用说明

本项目包含了多种编程语言的Hello World示例。以下是运行各个示例的基本指南：

## Python版本
```
cd python
python hello.py
```

## JavaScript版本
```
cd javascript
node hello.js
```

## Java版本
```
cd java
javac Hello.java
java Hello
```

## C++版本
```
cd cpp
g++ hello.cpp -o hello
./hello
```

这些示例都会输出"你好，世界！"和"Hello, World!"两行文本。
